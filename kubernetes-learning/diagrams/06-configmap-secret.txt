================================================================================
                    CONFIGMAP & SECRET INJECTION
================================================================================
Description: ConfigMaps store non-sensitive config, Secrets store sensitive data.
Both can be injected as environment variables or mounted as files.

┌─────────────────────────────────────────────────────────────────────────────┐
│                              POD                                             │

┌────────────┬──────────────────────────┬──────────────────────────┐
│            │ ConfigMap                │ Secret                   │
├────────────┼──────────────────────────┼──────────────────────────┤
│ Data       │ Plain text               │ Base64 encoded           │
│ Use Case   │ Config files, env vars   │ Passwords, tokens, certs │
│ Size Limit │ 1 MB                     │ 1 MB                     │
