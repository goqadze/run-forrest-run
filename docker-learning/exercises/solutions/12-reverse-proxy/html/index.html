Nginx Reverse Proxy Demo

This page is served by Nginx. The following routes are proxied to backend services:

/app1

Routes to App 1 service (port 3001)

/app2

Routes to App 2 service (port 3002)

/api/

Routes to API service with load balancing

/health

Nginx health check endpoint

Test with curl:

curl http://localhost/app1
curl http://localhost/app2
curl http://localhost/api/
